## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!

```bash
# Open in browser
open index.html
# or
start index.html  # Windows
```

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... more colors */
}
```

### Content
- Edit `index.html` to update your information
- Modify section content directly in the HTML
- Add or remove projects, skills, or experience items

### Animations
- Adjust animation speeds in `script.js`
- Modify Intersection Observer thresholds
- Customize timing in CSS transitions
